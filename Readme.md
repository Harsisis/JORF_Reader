# JORF reader
Ce programme a pour but de télécharger et lire les articles du Journal Officiel (JORF) afin d'en extraire divers informations (ex: sommaire)

Les articles du [Journal Officiel](https://www.legifrance.gouv.fr/) sont obtenues à travers la base [OPENDATA](https://echanges.dila.gouv.fr/OPENDATA/) de la [DILA](https://www.dila.gouv.fr/).

## Dépendences
La liste des dépendences uilisées pour ce projet est disponible dans le fichier: `requirements.txt`
Pour télécharger l'ensemble des dépendences requises pour le projet, executer la commande suivante:

```bash
pip install -r requirements.txt
```

## Prérequis
Ce script a été créé pour une utilisation avec:
- python: `3.13.9`
- pip: `25.3`
